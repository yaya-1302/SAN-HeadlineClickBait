Grafik Loss

Grafik loss dari data training

Terlihat bahwa nilai loss untuk data training menurun secara drastis pada epoch awal. Untuk nilai loss validation data, terlihat menurun secara perlahan, kedua nilai loss di akhir epoch terlihat menjadi datar dan jaraknya tidak terlalu jauh. Hal ini berarti model dapat mengeneralisasikan data yang belum pernah dipelajari dengan baik.

Grafik Accuracy

Grafik accuracy dari data training

Nilai accuracy untuk data train meningkat secara drastis di awal epoch, setelahnya peningkatan nilai accuracy terjadi secara perlahan dan peningkatannya tidak terlalu signifikan, yang mengindikasikan model tidak memerlukan terlalu banyak epoch dalam proses training. Selain itu, kurva validation data juga menunjukkan peningkatan dan accuracy nya selalu di atas accuracy data train hal ini menunjukkan bahwa model tidak hanya mengingat data train tetapi model belajar dari data tersebut, sehingga model memahami data validation yang diberikan.
